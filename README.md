Projeto desenvolvido em Java utilizando o padrão de projeto **Iterator** aplicado em um sistema de hamburgueria.

O objetivo é demonstrar como percorrer uma coleção de pedidos sem expor a estrutura interna da lista utilizada para armazenamento.

O padrão **Iterator** fornece uma maneira de acessar sequencialmente os elementos de uma coleção sem expor sua representação interna.

Ele permite:

- percorrer elementos de uma coleção
- encapsular a lógica de navegação
- desacoplar a estrutura de dados da forma de acesso
- facilitar manutenção e reutilização


A hamburgueria possui diversos pedidos registrados.

O sistema utiliza um Iterator para percorrer todos os pedidos cadastrados.
